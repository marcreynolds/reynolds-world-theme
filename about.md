This is a demonstration site for the Reynolds World Jekyll theme. You can use
this theme for your own Jekyll-powered blog or website.

The Reynolds World theme features automatic dark mode based on system
preferences, responsive design, and clean typography.

You can find the source code for this theme at GitHub:
[reynolds-world-theme](https://github.com/yourusername/reynolds-world-theme)

Learn more about Jekyll at [jekyllrb.com](https://jekyllrb.com/)
