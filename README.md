# Reynolds World Theme

A minimalist Jekyll theme inspired by Hey World's clean, content-focused
design.

## Features

- Clean, readable typography
- Minimalist design that puts content first
- Profile header with avatar
- Responsive layout
- RSS feed support
- SEO optimized

## Installation

Add this line to your Jekyll site's `Gemfile`:

```ruby
gem "reynolds-world-theme", path: "~/other-repos/reynolds-world-theme"
```

And add this line to your Jekyll site's `_config.yml`:

```yaml
theme: reynolds-world-theme
```

## Configuration

Add the following to your `_config.yml`:

```yaml
title: Your Name
author:
  name: Your Name
  email: your.email@example.com
  bio: A brief bio about yourself
  avatar: /assets/images/avatar.png
description: Your site description
```
